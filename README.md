Destroy enemy ships to keep yourself alive! 

A work in progress, but playable:
Clone to VSCode and open index.html with the Live Server extension.

How to play:
- Game starts right away.
- Steer the boat with W, A, D.
- Steer the cannon with arrow keys.
- Shoot with space. 

Features: 
- Increasing amount of enemy ships.
- Two enemy ship types; with and without missiles.
